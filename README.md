# MiniCalc — Static web calculator

A tiny, attractive calculator you can host on GitHub Pages.

Features
- Basic arithmetic: + - × ÷
- Parentheses, decimal numbers
- Percent, ± toggle, square root (√)
- Keyboard support
- Safe-ish client-side evaluation (sanitizes input)

Run locally
1. Open `index.html` in your browser (no server required).
2. Or run a quick local server:
   ```bash
   # Python 3
   python -m http.server 8000
   # then open http://localhost:8000
